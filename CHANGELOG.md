## [v0.7.0] - 2026-05-25

### Highlights

- Major CLI rewrite with Cobra, vault-backed credentials via omnitoken, and MkDocs documentation site

### Breaking

- CLI framework changed from go-flags to Cobra with new flag names ([`5bbe6f8`](https://github.com/grokify/aha-mcp-server/commit/5bbe6f8))

### Added

- Vault-backed credentials via omnitoken (1Password, Bitwarden, Keeper) ([`5bbe6f8`](https://github.com/grokify/aha-mcp-server/commit/5bbe6f8))
- CLI subcommands for all Aha! tools (get-idea, list-ideas, etc.) ([`5bbe6f8`](https://github.com/grokify/aha-mcp-server/commit/5bbe6f8))
- Aha omniskill package for composable MCP servers ([`c8862ac`](https://github.com/grokify/aha-mcp-server/commit/c8862ac))
- MkDocs documentation site with Material theme ([`182a51b`](https://github.com/grokify/aha-mcp-server/commit/182a51b))

### Changed

- Refactor skills package to use generic resource tool helper ([`7a5a0bc`](https://github.com/grokify/aha-mcp-server/commit/7a5a0bc))

### Fixed

- Resource leaks in API tools (missing `defer resp.Body.Close()`) ([`1ba3b78`](https://github.com/grokify/aha-mcp-server/commit/1ba3b78))
- JSON double-encoding bug in API responses ([`1ba3b78`](https://github.com/grokify/aha-mcp-server/commit/1ba3b78))
- Silent timestamp parsing failures now return clear errors ([`1ba3b78`](https://github.com/grokify/aha-mcp-server/commit/1ba3b78))

### Dependencies

- Add `github.com/plexusone/omnitoken` for vault-backed credentials ([`39c6006`](https://github.com/grokify/aha-mcp-server/commit/39c6006))
- Add `github.com/spf13/cobra` for CLI framework ([`39c6006`](https://github.com/grokify/aha-mcp-server/commit/39c6006))
- Bump `github.com/modelcontextprotocol/go-sdk` from 1.4.0 to 1.6.1 ([`39c6006`](https://github.com/grokify/aha-mcp-server/commit/39c6006))
- Bump `github.com/grokify/mogo` from 0.73.1 to 0.74.5 ([`39c6006`](https://github.com/grokify/aha-mcp-server/commit/39c6006))
- Bump Go version from 1.25.0 to 1.26.0 ([`39c6006`](https://github.com/grokify/aha-mcp-server/commit/39c6006))
